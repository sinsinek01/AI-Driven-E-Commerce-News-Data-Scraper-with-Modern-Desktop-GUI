# AI Data Miner Pro: Advanced Web Scraping & Automation Tool

AI Data Miner Pro is a high-performance Python-based automation tool designed to extract data from complex e-commerce platforms and news portals. Featuring a sleek, modern GUI built with CustomTkinter, it allows non-technical users to perform professional data mining with just a single click.

## 🚀 Key Features
* **Dynamic Content Scraping:** Utilizes Selenium to handle JavaScript-heavy websites and infinite scrolling.
* **Intelligent Data Structuring:** Automatically organizes extracted data into clean, analysis-ready Excel (XLSX) or CSV files using Pandas.
* **Modern Desktop GUI:** Built with a professional dark-themed interface for an enhanced user experience.
* **Real-time Console Logs:** Monitor the scraping process, including connection status and data count, directly through the integrated dashboard.
* **Standalone Executable:** Can be packaged as a .exe file, requiring no Python installation for the end-user.

## 🛠️ Tech Stack
* **Language:** Python 3.13
* **Automation:** Selenium WebDriver
* **Data Processing:** Pandas
* **UI Framework:** CustomTkinter

## 📥 How to Use
1. Clone the repository.
2. Install dependencies: `pip install selenium pandas customtkinter`.
3. Run `avci.py` to start the application.
4. Click 'Start Data Extraction' and wait for the 'Task Completed' status.
5. Open your generated report via the 'Open Excel Report' button.
